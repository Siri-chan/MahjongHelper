Rules


MahjongHelper uses Chinese Official International Rules for Mahjong (MCR). This is just because these are the rules I use to play with my friends.

MahjongHelper is open-source, so you can fork it, and change it, if you want to use different rules.

Eventually there will be a first-party copy of these rules here.

MahjongEd also have a PDF describing this set of rules. We also host a mirror of this PDF.
